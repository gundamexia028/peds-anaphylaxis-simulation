# V1.2.6d 心肺骤停未CPR死亡事件记录修正说明

本版本基于 V1.2.6c，不改变25分标准主路径，仅修正心肺骤停后的危重分支终止逻辑。

## 一、修正背景

V1.2.6c 已能在心肺骤停时显示客观临床表现，但如果心肺骤停后受试者下一步没有选择 CPR，系统仍可能继续允许其他操作。

根据当前考核逻辑，患儿已进入心肺骤停状态后，普通病房护士的即时关键反应应为 CPR。若下一步未选择 CPR，应判定为严重延误复苏并记录死亡事件。

## 二、新增规则

心肺骤停状态触发后：

- 下一步选择 CPR：进入复苏分支；
- 下一步选择任何非 CPR 操作：记录死亡事件并终止情景；
- 选择“时间流逝”且未 CPR：记录死亡事件并终止情景。

## 三、新增/强化字段

- death_event
- death_after_arrest_without_cpr
- cpr_omission_death
- death_time_sec
- death_reason
- outcome_class = death_after_cardiac_arrest_without_cpr

## 四、对研究分析的影响

该逻辑不改变25分标准主路径。死亡事件属于条件性危重分支结局，可用于分析心肺骤停后的复苏反应能力。
