# V1.2.1 2025共识对齐与步骤规划说明

本版本在 V1.2.0 研究质控与分析版基础上，依据 2025 年《严重过敏反应诊断和临床管理专家共识》的临床表现、初始治疗流程和儿童肾上腺素剂量规则进行优化，仅用于护理教学、培训和科研模拟，不用于临床诊疗决策。

## 一、病情进展分级展示

系统新增“病情进展分级”卡片，按模拟状态动态显示：

| 分级 | 系统展示重点 |
|---|---|
| I级 | 早期皮肤黏膜/前驱期：瘙痒、潮红、风团样皮疹、轻度咳嗽或烦躁 |
| II级 | 呼吸、胃肠或早期循环受累：咳嗽、胸闷、喘息、喉部发紧、呕吐、SpO₂下降或血压下降 |
| III级 | 严重呼吸受累/循环衰竭：喉鸣、发绀、明显低氧、低血压、末梢灌注差、意识改变 |
| IV级 | 心跳或呼吸骤停：需立即就地CPR并启动高级生命支持 |

## 二、儿童肌注肾上腺素剂量规则

系统模拟对象调整为 2–11 岁儿童。肌注肾上腺素剂量核对规则为：

- 目标剂量：0.01 mg/kg；
- 单次最大剂量：0.3 mg；
- 允许 ±0.01 mg 的四舍五入容差；
- 低于目标剂量：判定为低剂量/操作无效；
- 高于目标剂量但未超过 0.3 mg：判定为剂量不准确/用药安全缺陷；
- 超过 0.3 mg：判定为严重用药安全事件。

## 三、抢救操作步骤规划限制

V1.2.1 对部分操作增加前置条件，避免不符合抢救逻辑的路径。例如：

| 操作 | 前置逻辑 |
|---|---|
| 肌注肾上腺素 | 需先停用可疑药物并呼救，且病情符合或高度疑似严重过敏反应 |
| 晶体液补液 | 需确认静脉通路，并有血压/灌注或循环不稳定依据 |
| 抗组胺药/糖皮质激素 | 只能作为肌注肾上腺素后的辅助治疗，不得替代一线治疗 |
| 雾化支气管扩张剂 | 肌注肾上腺素后仍有喘息/支气管痉挛时使用 |
| 雾化肾上腺素 | 肌注肾上腺素后仍有喉鸣/上气道水肿表现时作为辅助处理 |
| 重复肌注肾上腺素 | 首次肌注后 5–15 min 复评仍未缓解时才开放 |
| 呼叫急诊/PICU/ICU | 两次肾上腺素及补液后仍无缓解，或病情进入难治性路径时开放 |
| SBAR交接 | 至少完成复评并初步稳定后开放 |

## 四、评分结构

满分仍为 20 分。评分更突出救命核心操作和流程完整性：

- 去除可疑诱因与保留抢救通路；
- 呼救和团队启动；
- ABC评估；
- 气道开放和给氧；
- 监护、血压和灌注评估；
- 早期肌注肾上腺素及剂量正确性；
- 静脉通路确认和循环支持；
- 喘息/支气管痉挛的辅助处理；
- 复评、家属沟通、SBAR交接与后续观察安排。

## 五、导出字段影响

V1.2.1 在完整 JSON 和导出 CSV 中继续保留 V1.2.0 的研究字段，并新增/强化以下分析意义：

- `epinephrine_excess_dose`：高于目标剂量但未超过儿童单次上限；
- `iv_access_time_sec`：确认/建立静脉通路时间；
- `repeat_epinephrine_time_sec`：重复肌注肾上腺素时间；
- `advanced_support_time_sec`：呼叫急诊/PICU/ICU高级支持时间；
- `final_progression`：最终病情进展分级说明。

## 六、注意事项

本系统用于模拟培训和科研数据采集。任何临床处置均应以所在医院制度、医嘱、抢救流程和最新指南为准。
