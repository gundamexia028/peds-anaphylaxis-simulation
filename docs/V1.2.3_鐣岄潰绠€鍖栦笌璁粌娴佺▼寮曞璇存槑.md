# V1.2.3 界面简化与训练流程引导说明

## 更新目标

本版本在 V1.2.2 基础上进一步优化受试者端界面，避免页面文字对考试模式产生过度提示，同时保留系统内部的指南对齐逻辑、动态病情进展、评分规则和训练模式标准流程引导。

## 主要调整

1. 删除受试者端“判断提示”显示。
2. 删除受试者端“病情进展分级”显示。
3. 顶部运行信息不再显示分级，仅显示时间、得分、复评次数等运行信息。
4. “当前临床表现”继续保留，但仅呈现症状/体征，不直接提示诊断判断。
5. 操作按钮全部保持可选择，不再灰显或限制点击。
6. 操作按钮文字简化，例如“停用可疑药物”“呼救”“ABC评估”“开放气道给氧”“肌注肾上腺素”“快速补液”等。
7. 标准处置流程仅在训练模式中作为提示和偏离提醒；考试模式不显示标准流程提示。
8. 内部评分、病情进展、剂量判定、特殊路径触发和数据导出字段保持不变。

## 设计原则

- 考试模式：不提示诊断、不提示分级、不限制操作，尽量记录真实处置路径。
- 训练模式：可提示标准流程，但不锁定操作按钮。
- 数据分析：后台仍保留病情分级、标准流程偏离、给药时间、剂量正确性和关键步骤完成情况等字段。
