# V1.2.6e 生命体征恢复范围修正说明

本版本基于 V1.2.6d 继续修正，25分标准主路径不变。

## 修正原因

在正常标准路径中，完成肌注肾上腺素、快速扩容、复评、雾化支扩和糖皮质激素后，多个“改善效果”连续叠加，可能出现不合理生命体征，例如：

- SpO₂ 100%；
- HR 64/min；
- RR 5/min；
- BP 160/109 mmHg。

这些数值不符合“抢救后趋于稳定”的普通恢复逻辑，属于生命体征恢复算法过度矫正。

## 本次修正

1. 糖皮质激素仍计入标准路径5分，但不再造成即时生命体征大幅改善；
2. 糖皮质激素定位为辅助治疗和剂量计算考核，不替代肾上腺素、给氧、通气或扩容；
3. 标准恢复分支加入生理显示边界，避免 HR/RR 过低、BP 过高、SpO₂ 过度完美；
4. 心肺骤停、死亡和复苏后ROSC分支仍保留独立显示逻辑；
5. 危重分支、CPR未及时执行死亡事件记录逻辑不变。

## 标准恢复显示原则

正常标准路径恢复过程中，生命体征应表现为“趋于稳定但仍需观察”，而不是完全正常或过度矫正。

建议显示范围：

- SpO₂：约95%–98%；
- HR：约100–140/min，低龄儿童可略高；
- RR：约20–36/min，低龄儿童可略高；
- BP：接近年龄对应正常范围，避免明显高血压。
