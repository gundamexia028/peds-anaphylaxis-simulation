# V1.1.3 登记优化与编码规则说明

## 标准化院区

- 锦江院区：JJYQ
- 眉山院区：MSYQ
- 高新院区：GXYQ

## 标准化科室

- 呼吸科：HXK
- 感染科：GRK
- 肾脏科：SZK
- 血液科：XYK
- 心血管科：XXGK
- 神经科：SJK
- PICU：PICU

## 参与者编号

系统根据“院区代码 + 科室代码 + 姓名首字母 + 防重复后缀”生成参与者编号。

示例：眉山院区、呼吸科、王思席（WSX）可生成为：

`MSYQHXKWSX-XXXX`

其中 `XXXX` 为系统自动生成的防重复后缀，用于降低相同院区、相同科室、相同姓名首字母导致的编号重复风险。

## 前台删除字段

- 培训批次/项目编号：前台删除，后台默认为空。
- 第几次尝试/测试：前台删除，后台默认 attempt_no=1。

## 评估阶段

前台固定为三个阶段：

1. 基线评估
2. 模拟培训
3. 培训后考核
