# V1.1.1 数据导出说明

## 1. 训练汇总 CSV

每次训练一行，适合直接导入 Excel、SPSS、R 或 Python。主要字段包括：

- 参与者编号、单位、科室
- 运行模式、情景脚本
- 年龄、体重
- 得分、原始分、扣分、得分率
- success/failure/timeout
- 关键步骤完成时间
- 肾上腺素目标剂量、实际剂量、剂量状态
- 剂量不足/超量标记
- 错误操作次数、高危错误次数
- 最终生命体征
- 过程性安全缺陷和缺失关键动作
- 完整操作顺序

## 2. 操作明细 CSV

每个操作事件一行，适合分析受试者的操作路径。主要字段包括：

- 参与者编号、单位、科室、session_id
- 操作序号
- 操作发生时间
- 操作名称
- 事件类型：操作 / 扣分 / 剂量判定
- 得分、扣分、剂量、目标剂量
- 结果、原因
- 最终得分和情景结局

## 3. 完整 JSONL

每次训练一行完整 JSON 报告。适合长期归档和后续深度分析。普通 Excel 不适合直接查看 JSONL，建议用 Python/R 读取。
