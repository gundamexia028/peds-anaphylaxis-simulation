# V1.1.4 评估阶段流程锁定说明

本版本在 V1.1.3 登记优化版基础上新增“按评估阶段自动锁定流程”功能，用于避免基线评估、模拟培训和培训后考核数据混杂。

## 锁定规则

| 评估阶段 | 自动运行模式 | 自动病例脚本 |
|---|---|---|
| 基线评估 | 考试模式 | 初始病例 |
| 模拟培训 | 训练模式 | 初始病例 |
| 培训后考核 | 考试模式 | 变体病例 Variant A |

受试者完成登记后，系统依据其选择的评估阶段自动确定运行模式与病例脚本。受试者不能再自行选择运行模式或病例脚本。

## 新增导出字段

管理员导出的汇总表、操作明细表和完整 JSONL 中新增以下流程字段：

- workflow_mode
- workflow_script_role
- workflow_display
- workflow_locked

## 保留内容

本版本保留 V1.1.3 的登记逻辑，包括院区下拉、科室下拉、姓名首字母、自动生成参与者编号、护理层级、工作年限、既往培训经历、评估阶段等字段。
