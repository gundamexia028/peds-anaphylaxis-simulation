# V1.1.2 多中心研究字段说明

## 新增字段

- campus：院区/中心
- department_type：科室类型
- nurse_level：护理层级
- years_experience：工作年限
- professional_title：职称
- education_level：最高学历
- prior_anaphylaxis_training：是否接受过过敏反应/过敏性休克培训
- prior_simulation_experience：是否参加过模拟/虚拟仿真培训
- real_case_experience：是否处理过真实过敏反应病例
- training_batch：培训批次/项目编号
- assessment_phase：评估阶段
- attempt_no：第几次尝试/测试

## 数据流向

1. 登录登记页填写；
2. 写入单次训练 report 的 session 字段；
3. 保存到 Supabase 的 full_report JSON；
4. 管理员后台导出汇总 CSV 和操作明细 CSV 时展开为独立字段。

## 适用分析

- 多院区比较
- 不同科室类型比较
- 不同护理层级比较
- 工作年限影响因素分析
- 既往培训经历与操作表现分析
- 基线评估与训练后评估比较
