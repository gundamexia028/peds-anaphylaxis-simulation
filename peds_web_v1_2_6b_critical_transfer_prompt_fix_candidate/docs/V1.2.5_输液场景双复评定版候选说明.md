# V1.2.5 输液场景双复评定版候选说明

## 一、改进目的

本版本用于解决 V1.2.4 系列中发现的核心问题：

1. 输液过程中发生过敏反应，不应设置“建立静脉通路”为标准得分项。
2. 快速补液需要输入具体容量，不能只点按钮。
3. 复评应分为“第一次复评”和“第二次复评”，不能用同一个按钮重复计分。
4. 再次肌注肾上腺素不是所有人必做，应作为条件性特殊路径。
5. 前台不应显示内部英文状态或病情分级提示。
6. 未进行雾化支扩时，不应显示“无异常”。

## 二、标准路径

总分仍为 20 分：

- 停用可疑药物并保留通路：2分
- 呼救：2分
- ABC评估：1分
- 开放气道给氧：2分
- 体位管理：1分
- 连接监护：1分
- 测血压：1分
- 肌注肾上腺素：4分
- 快速补液：2分
- 第一次复评：1分
- 雾化支扩：1分
- 第二次复评：1分
- 告知家属 + SBAR交接：1分

## 三、复评逻辑

### 第一次复评

有效条件：

- 已完成有效肌注肾上腺素；
- 已完成有效快速补液；
- 点击“第一次复评”。

无 5–15 min 硬性时间卡点。

### 第二次复评

有效条件：

- 已完成第一次有效复评；
- 尚未告知家属；
- 点击“第二次复评”。

第二次复评不强制绑定雾化支扩，但如果未做雾化支扩，临床表现仍保留“咳嗽/轻微喘息”。

## 四、剂量与容量规则

### 肾上腺素

- 0.01 mg/kg；
- 儿童单次最大 0.3 mg；
- 正确范围：目标剂量 ±0.01 mg；
- >0.3 mg 判定为严重用药安全事件。

### 快速补液

- 10–20 ml/kg；
- 单次最大 500 ml；
- 需输入具体 ml 数；
- 拔除静脉通路后补液无效。

## 五、特殊路径

以下路径不进入普通20分：

- 再次肌注肾上腺素；
- 联系高级支持；
- CPR；
- 雾化肾上腺素；
- 抗组胺药；
- 糖皮质激素。

系统会根据条件记录其是否适用、是否提前、是否非必要或是否替代了一线治疗。

## 六、导出字段新增/强化

管理员导出汇总中会展开：

- fluid_bolus_volume_ml
- fluid_min_ml
- fluid_max_ml
- fluid_bolus_valid
- first_reassessment_time
- second_reassessment_time
- repeat_epinephrine_time
- advanced_support_time
- cpr_time
- dose_high_epi
- serious_medication_error（完整JSON中保留）
