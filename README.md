# 儿科药物诱发过敏反应动态分支虚拟仿真训练系统｜V1.2.7

## 本版定位

V1.2.7 在 V1.2.6g 基础上完成两项结构性更新：

1. **普通路径升级为100分模块边界评分**：抢救启动20分、初始支持与评估20分、首次肌注肾上腺素25分、肾上腺素后标准处理20分、复评交班沟通15分。
2. **既往经历问题后置**：登记页不再显示“是否接受过过敏反应/过敏性休克培训”“是否参加过模拟/虚拟仿真培训”“是否处理过真实过敏反应病例”。三项问题仅在基线评估操作完成并锁定评分后显示一次，提交后基线阶段才正式结束。

## 三阶段流程锁定

| 阶段 | 模式 | 病例脚本 |
|---|---|---|
| 基线评估 | 考试模式 | 初始病例 |
| 模拟培训 | 训练模式 | 初始病例 |
| 培训后考核 | 考试模式 | Variant A |

受试者不能自行选择运行模式、病例脚本或随机种子。

## V1.2.7普通路径评分

| 模块 | 名称 | 分值 | 评分边界 |
|---|---|---:|---|
| 模块1 | 抢救启动模块 | 20 | 停止输液、呼救；0–60秒满分，61–120秒半分，超过120秒不补分 |
| 模块2 | 初始支持与评估模块 | 20 | ABC评估、吸氧、体位、监护、测血压；首次有效肌注肾上腺素前满分，肌注后至第一次有效复评前半分，第一次有效复评后不补分 |
| 模块3 | 首次肌注肾上腺素关键模块 | 25 | 正确完成首次肌注肾上腺素剂量输入后计分，并作为前后流程分界点 |
| 模块4 | 肾上腺素后标准处理模块 | 20 | 快速补液、第一次复评、雾化支扩、糖皮质激素 |
| 模块5 | 复评、交班与沟通模块 | 15 | 第二次复评、家属沟通、SBAR交接；完成后普通考核结束 |

模块边界用于评分归属和延迟判定，不作为绝对操作锁。早期抢救允许合理并行，肾上腺素后进入顺序化评分。

## 条件分支

再次肌注肾上腺素不属于普通主线得分项。前端默认隐藏，仅当第一次有效复评后系统判定存在改善不足或恶化时才显示。普通路径未再次肌注不扣分。

气道梗阻、球囊面罩加压给氧、高级生命支持、CPR仍作为条件性危重分支单独记录。
