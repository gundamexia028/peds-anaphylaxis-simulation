# 儿科药物诱发过敏反应动态分支虚拟仿真训练系统｜V1.2.6a 糖皮质激素计分与气道分支候选版

本版本在 V1.2.5 输液场景双复评版基础上继续升级，用于支撑：

> 基于动态分支虚拟仿真平台的多院区儿科护士药物诱发过敏反应识别与初始处置能力现状、影响因素及培训效果研究

## 本版定位

V1.2.6a 是“代码候选版”，重点完成以下逻辑：

1. 标准路径从 20 分升至 **25 分**；
2. 删除“抗组胺药”按钮；
3. “糖皮质激素”纳入标准路径，计 **5 分**，需输入剂量；
4. 糖皮质激素必须在有效肌注肾上腺素和快速扩容之后使用；
5. 未及时肌注肾上腺素不作为立即停止条件；
6. 前7项基础抢救操作期间生命体征缓慢演化；
7. 到达第8关键节点后仍未肌注肾上腺素，启动明显生命体征波动；
8. 气道梗阻、球囊面罩加压给氧、高级生命支持、CPR作为条件性危重分支单独记录，不混入25分主分。

本版仍沿用 Supabase `training_records` 表，不强制新增数据库顶层字段；新增研究字段保存在 `full_report` JSON 中，并由管理员后台导出 CSV 时展开。

## 药物和容量判定

### 肌注肾上腺素

- 1:1000 肾上腺素；
- 0.01 mg/kg；
- 儿童单次最大 0.3 mg；
- 正确判定范围：目标剂量 ±0.01 mg；
- >0.3 mg 判定为严重用药安全事件。

### 快速扩容

- 晶体液 10–20 ml/kg；
- 单次最大 500 ml；
- 点击“快速扩容”后需输入 ml 数；
- 拔除静脉通路后再扩容，判定为流程矛盾。

### 糖皮质激素

- 当前默认考核药物：甲泼尼龙；
- 参考剂量：1–2 mg/kg；
- 单次最大 40 mg；
- 必须在有效肌注肾上腺素和有效快速扩容之后使用；
- 输入剂量正确且时机正确才作为完整标准路径。

糖皮质激素5分拆分：

| 评价点 | 分值 |
|---|---:|
| 使用时机正确：有效快速扩容后 | 2 |
| 未替代一线治疗：已有效肌注肾上腺素 | 1 |
| 剂量输入正确 | 2 |

## 病情动态演化

V1.2.6a 采用“步骤节点 + 时间”双触发逻辑：

- 前7项基础抢救动作期间，未肌注肾上腺素导致的生命体征变化较缓；
- 到达第8关键节点后，如果仍未完成有效肌注肾上腺素，判定为关键一线药物延误；
- 随后启动明显生命体征波动，包括SpO₂下降、HR/RR升高、BP下降、喘息/喉鸣/血管性水肿风险增加；
- 持续未处理时可进一步触发气道梗阻、球囊面罩加压给氧、高级生命支持或CPR条件。

## 条件性危重分支

以下路径不纳入25分主分，单独记录：

- 再次肌注肾上腺素；
- 联系高级生命支持；
- 球囊面罩加压给氧；
- 雾化肾上腺素；
- CPR。

这些字段主要用于次要或探索性分析，不影响培训前后主分比较。
